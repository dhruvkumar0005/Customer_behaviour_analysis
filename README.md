# Customer_behaviour_analysis
Data Analytics Project Showcasing Customer Behavior analysis using Python, SQL, and Power BI.


Project Steps

1. Data Loading
Imported dataset using Pandas.
Verified structure and data types.
Inspected missing values and summary statistics.

2. Data Cleaning
Handled missing values in Review Rating using category-wise median.
Standardized column names.
Renamed columns for better readability.
Removed redundant columns.
Created new derived features:
Age Group
Purchase Frequency (Days)

3. Exploratory Data Analysis (EDA)
Performed analysis on:
Customer age distribution
Purchase amount trends
Product category popularity
Review ratings
Discount impact on purchases
Purchase frequency patterns

4. SQL Integration
Connected Python with MySQL using SQLAlchemy.
Uploaded cleaned dataset to MySQL database.
Executed SQL queries for business insights.

5.Dashboard
An interactive Power BI dashboard was created to visualize:
Total Revenue
Average Purchase Amount
Customer Demographics
Category-wise Sales
Purchase Frequency Analysis
Discount Usage Trends
Customer Rating Distribution
